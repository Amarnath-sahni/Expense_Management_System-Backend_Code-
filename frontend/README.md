# Expense Management System - Frontend 🚀

A modern and responsive frontend application for managing personal expenses.
Built using **React 19, Vite,Context, and Tailwind CSS**, providing a smooth user experience with authentication, expense tracking, analytics visualization, and report generation.


---

# ✨ Features

## 🔐 Authentication & User Management

* User registration and login functionality.
* Two-step email verification using OTP.
* JWT authentication flow.
* Protected routes for authenticated users.
* Secure user session handling.

---

## 💰 Expense Management

* Add, update, delete, and view expenses.
* Expense categorization:

  * 🍔 Food
  * ✈️ Travel
  * 🛒 Shopping
  * 💳 Bills
  * 📦 Others
* Expense history management.
* Monthly expense tracking.

---

## 📊 Dashboard & Analytics

* Interactive expense dashboard.
* Visual representation of spending patterns.
* Category-wise expense analysis.
* Dynamic charts using Recharts.
* Monthly expense summaries.

---

## 📄 Report Generation

* Export expense reports as PDF.
* Generate downloadable documents.
* File download management using File Saver.

---

## 🤖 AI Integration

* Integrated Google GenAI for AI-powered features.
* Smart assistance and intelligent expense insights.

---

## 🎨 UI & User Experience

* Responsive design for desktop and mobile.
* Modern UI with Tailwind CSS.
* Smooth animations using Framer Motion.
* Reusable React components.
* Icon support using Lucide React and React Icons.
* Client-side routing with React Router.

# ⚙️ Installation & Setup

### Clone Repository

```bash
git clone <repository-url>
```

### Navigate to Frontend

```bash
cd frontend
```

### Install Dependencies

```bash
npm install
```

### Environment Variables

Create `.env` file:

```env
VITE_API_URL=http://localhost:5000/api
VITE_GOOGLE_GENAI_KEY=your_api_key
```

### Run Development Server

```bash
npm run dev
```

Application runs at:

```bash
http://localhost:5173
```
